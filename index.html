Jila

An open-source framework allowing communities with endangered languages to build engaging apps that help to preserve their culture

What is Jila?

Jila began as the core of Yawuru Ngan-ga, a language app from the Yawuru community of Broome, Western Australia. From there the team transformed it into an almost turnkey solution for communities who want to create their own language applications without requiring a large financial outlay, or a team of developers to maintain the content.

The framework consists of two components, a cloud-ready administration console for the curation of the dictionary and content, and a cross-platform mobile application built using Apache Cordova. More information can be found on their respective Github repositories.

How do I get involved?

Fork the repositories and have a go at building your own version of the app! If you publish any apps based on the framework please let us know.

If you make any enhancements or additions to the framework we'd really appreciate if you're able to submit them as pull requests. That way all of the communities building apps based on Jila can benefit collectively.
